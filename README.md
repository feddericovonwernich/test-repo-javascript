# Test Repo JavaScript

A Node.js project demonstrating JavaScript/TypeScript best practices and testing with Jest.

## Description

This repository tests the scorecards system with a JavaScript-specific setup, including:
- Jest test framework with `__tests__` directory
- npm package management
- GitHub Actions for Node.js CI
- Modern JavaScript/ES6+ syntax

## Installation

```bash
npm install
```

## Testing

```bash
npm test
```

## Running

```bash
npm start
```

## Project Structure

- `src/` - Source code
- `__tests__/` - Jest test suite
- `package.json` - Node.js dependencies and scripts
